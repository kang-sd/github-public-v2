십성 해석 정리표

십성 많을 경우 없을 때 다른 십성과 관계
👬🤝 비겁
(비견+겁재)		 • 신왕해져 인성 도움 불필요
• 공부엔 관심 적고 친구와 어울림
• 윗사람 간섭 거부, 독립성 강함
• 자수성가, 개인사업 적합
• 추진력·고집 강 → 성공의 동력
• 중도 포기만 피하면 성취 가능
• 낭비벽 심, 재물 소모 큼
• 자아 강하고 인정 욕구 강함
• 사람 많으나 의존적 → 고독감 		• 주변 기대에 눌려 전전긍긍
• 작심삼일, 결단력 부족
• 우유부단, 소극적 		👬 비겁(비견·겁재) = 나·동료 / 고집·경쟁
• 일지 비견 + 귀인성(천을·천덕·월덕·문창) → 예술/엔터 성취
🎨🎤 식상
(식신+상관)		 • 총명·아이디어 발휘
• 예체능·특수기술·교육·시민운동 적합
• 자유분방, 구속 싫어함
• 일 벌림 많으나 마무리 부족
• 말만 앞서거나, 일에 미쳐 몸 혹사
• 좋아하는 일에서 희열·성취
• 과다하면 결혼·가정 등한시 		• 외부 간섭·통제에 순응
• 억울감·우울감 누적
• 순간 욱하고 후회 		🎨 식상(식신·상관) = 창조·표현 / 자유·예술
• 식신+편인: 편인 억압 시 식신 약화 → 재성 오면 회복
• 식상혼잡: 집중력 약화, 직업·사업 변화 심함
• 식신+편관: 식신이 편관 제어 → 제살(制殺)
💰📦 재성
(정재+편재)		 • 재물 집착·사업확장
• 돈·여자 문제로 재앙 가능
• 인성 극 → 공부·문서 약화, 가출 성향
• 일간 약 → 재성 태과 시 건강 문제, 과로
• 소유욕·낭비벽·집착 등 유형 다양
• 집착은 장점, 낭비는 위험 		• 노력해도 결과 불만족
• 비교·열등감
• 비효율적, 에너지 소모 큼 		💰 재성(정재·편재) = 재물·소유 / 욕심·투자
• 재성 혼잡: 돈 들어와도 빠져나감
• 정재→정관: 원칙·보수적, 경영·재무 적합
• 편재→정관: 편법재물 → 불만족·구설
• 정재→편관: 안정 후 명예 추구 → 재물 손실
• 편재→편관(일간 강): 지도자, 추진력
• 편재→편관(일간 약): 불륜·건강·자식 문제
• 정재·편재가 인성을 억제 → 계약·학업 손실/현실 감각 부여
⚖️🛡️ 관성
(정관+편관)		 • 인내·자제·규범의 힘
• 신약이면 질병·스트레스 많음
• 자신감 약, 타인 의존
• 태생적으로 예민·예술적
• 마음 먹으면 누구보다 독함
• 스스로 죄여오는 느낌, 짜증·불만 多 		• 능력 무시, 책임만 과중
• 자유 갈망·현실 불만
• 탈출 어려움, 불만 내재 		⚖️ 관성(정관·편관) = 규율·권위 / 책임·통제
• 정관 2↑ → 조직 승진 유리
• 정관+비견 → 가업·관직
• 정관+편재 → 제조업·상공업
• 정관+겁재 → 이성 문제
• 편관+편인 → 예술·엔터테인먼트
• 편관+정인 → 순수예술·전통예술
• 편관+겁재 → 보증·채무 손실
📚🧘 인성
(정인+편인)		 • 전문성·자격증·학문으로 길 열림
• 생각 많음·소심·의존적
• 표현력 강하나 추진력 부족
• 애정결핍·고립감
• 아는 것 많으나 설득력 약
• 행동하는 지식인·전문가 될 것 		• 미래지향적이지만 실행력 부족
• 불만 많음, 환경 탓
• 의욕은 있으나 실천 부족 		📚 인성(정인·편인) = 학문·보호 / 전문성·사색
• 편인: 이별·고독·예술·종교성
• 편인 과다: 계모 슬하, 허약, 가정 불화
• 편인+관살 → 부귀
• 편인+여명 과다 → 남편 대신 생존경쟁
• 정인 강 → 보수적, 실행력 약화
